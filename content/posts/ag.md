`ack`より、'better'と主張している[ag(silver search)](https://github.com/ggreer/the_silver_searcher)を入れました。

<!--more-->

## インストール
```sh
$ brew install ag
```

## 利用方法
```sh
$ ag [options] {pattern} [{directory}]
```

早いし、結果表示も見やすく、とても良いです。
